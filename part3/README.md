# Phonebook Full Stack Application

Full stack phonebook application built with React frontend and Node.js/Express backend.

## Project Structure

```
part3/
├── src/
│   ├── index.js              # Backend Express server
│   ├── main.jsx              # Frontend React entry point
│   ├── App.jsx               # Main React component
│   ├── index.css             # Frontend styles
│   ├── components/           # React components
│   │   ├── PersonForm.jsx    # Add person form
│   │   ├── PersonList.jsx    # Display persons list
│   │   └── Notification.jsx  # Notification messages
│   └── services/
│       └── personService.js  # API service for backend communication
├── dist/                     # Frontend production build (generated by `npm run build`)
├── package.json              # Dependencies and scripts
└── vite.config.js            # Frontend build config
```

## Prerequisites

- Node.js (v18+)
- npm

## Installation

```bash
npm install
```

## Development

### Run Backend and Frontend Separately

**Open two terminals:**

**Terminal 1 - Backend (Port 3001):**
```bash
npm run dev:backend
```

**Terminal 2 - Frontend (Port 5173):**
```bash
npm run dev:frontend
```

The frontend will open automatically at `http://localhost:5173`

### Backend API Endpoints

- `GET /api/persons` - Get all persons
- `GET /api/persons/:id` - Get a specific person
- `POST /api/persons` - Add a new person
- `DELETE /api/persons/:id` - Delete a person
- `GET /info` - Get info page with person count
- `GET /` - Root endpoint

## Building for Production

### Build Frontend
```bash
npm run build
```

This creates a `dist/` folder with the production-ready frontend build.

### Run Production Server

```bash
npm start
```

The server will start on port 3001 (or use the PORT environment variable).
- Backend API will be available at `/api`
- Frontend will be served from `/`

## Testing

### Test with curl (Backend)

```bash
# Get all persons
curl http://localhost:3001/api/persons

# Get info page
curl http://localhost:3001/info

# Add new person
curl -X POST http://localhost:3001/api/persons \
  -H "Content-Type: application/json" \
  -d '{"name":"John Doe","number":"123-456-7890"}'

# Delete person
curl -X DELETE http://localhost:3001/api/persons/1
```

### Test Frontend
1. Open http://localhost:5173 in browser
2. Add new contacts using the form
3. Delete contacts using delete button
4. Check browser console and Network tab for API requests

## Environment Variables

Create a `.env` file in the root directory (optional):

```
PORT=3001
NODE_ENV=development
```

## Notes

- The phonebook data is stored in `osoby.json` (in-memory during runtime)
- Changes are not persisted to disk - restart server to reset data
- Frontend uses Vite for development and builds
- Backend uses Express.js for API and static file serving
- CORS is enabled to allow frontend to access backend API during development
